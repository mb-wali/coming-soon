# coming soon
A sample coming-soon page.

# Pull image
```docker pull mbwali/coming-soon```

# Build image 
```docker build -t coming-soon:latest .```

# Run
```docker run -d -p 8080:8080 coming-soon:latest```

Once running, visit http://127.0.0.1:8080 in your browser.

- HTML
